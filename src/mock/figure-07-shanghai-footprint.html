How a giant city absorbs land

Some giant cities grow so fast that one fixed map frame is enough to make the point. Shanghai is a good candidate because the visual story is not subtle.

This mock holds the camera still and layers five morphological footprints on top of one another. The effect should be spatial shock, not a case study full of annotation.

The useful thing about a scene like this is that it does not need much explanation. If the geometry is clean, the land absorption is the argument.
